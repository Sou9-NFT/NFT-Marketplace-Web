Download dependencies
```
composer install
```

How to start a server with Symfony
```
symfony server:start 
symfony serve
```

> [!WARNING]
> change the path of the assets in the twig files example:
> 
> `css/style.css` -> `front_office/css/style.css` OR `back_office/css/style.css`.
